Solo project: Spending Habits

Website that tracks a user’s spending.
Spending will be tracked by ‘Expenses’ and these will be manually added.
Each expense has a detail and date.
Expenses will have CRUD functions.
Clicking on the date will show expenses in the same date.

Stack list:

Front-end: Javascript and Bootstrap
Back-end: Python and MySQL

GitHub link: https://github.com/jonny-txt/Spending_Habits
